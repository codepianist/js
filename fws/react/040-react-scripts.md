# 040-react-scripts.md

## Scripts
```json
  "scripts": {
    "start": "react-scripts start", // starts react application
    "build": "react-scripts build", // create optimized version of application on build/ folder
    "test": "react-scripts test", //
    "eject": "react-scripts eject" // generates config/ and scripts/ folders u can directily customize the internals of React
  },
```
